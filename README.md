# Casebriefr

**A professional legal case briefing tool for law students and legal professionals.**

🌐 **https://casebriefr.com**

---

## Overview

Casebriefr is a clean, printable HTML template for creating structured legal case briefs. It guides you through the essential components of case analysis with helpful prompts and a professional layout.

## Features

- ✅ **Structured Sections** — All essential case brief components (Facts, Issues, Holding, Rule, Reasoning, etc.)
- ✅ **Print-Optimized** — Clean formatting that looks great on paper or as PDF
- ✅ **Fillable Fields** — Type directly in your browser, then print or save as PDF
- ✅ **Professional Design** — Clean, readable layout suitable for class or court

## How to Use

1. Go to https://casebriefr.com/template.html
2. Fill in the case information in each section
3. Click "Print / Save as PDF" when finished
4. Save digitally or print for your records
